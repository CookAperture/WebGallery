# WebGallery

## Basic Features
- [ ] Gallery of All Pictures
- [ ] Database for All Pictures
- [ ] Upload of Picture(s)
- [ ] Download of Picture

## Advanced Features
- [ ] User Accounts
- [ ] Database for User Accounts
- [ ] Show Pictures only for Logged User
- [ ] Searchbar for Pictures
- [ ] Download of Pictures
